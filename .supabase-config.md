# Supabase CLI Setup (Optional)

Si quieres usar Supabase CLI desde la terminal:

## 1. Instalar Supabase CLI
```bash
# macOS con Homebrew
brew install supabase/tap/supabase

# O con npm
npm install -g supabase
```

## 2. Login en Supabase
```bash
supabase login
```
Esto abrirá el navegador para autenticarte.

## 3. Enlazar tu proyecto
```bash
supabase link --project-ref tu-project-ref
```
El project-ref lo encuentras en: Supabase Dashboard → Settings → General → Reference ID

## 4. Ejecutar SQL
```bash
supabase db push --file supabase-schema.sql
```

PERO la forma más fácil es usar el SQL Editor en el dashboard web.
